# FitManager - Project Overview

## Project Name
FitManager

## Project Type
Private fitness club management system for one client who owns a gym.

This project is NOT a SaaS platform.
It is NOT multi-tenant.
It is made for one fitness club owner and his members.

## Main Goal
The goal of FitManager is to help a fitness club owner manage:

- Members
- Subscriptions
- Payments
- Coaches
- Attendance
- Member accounts
- AI reminder messages

## Users
The application has two roles:

### Admin
The gym owner or manager.

### Member
A gym client who can register and access his personal dashboard.

## Main Modules

- Public website
- Authentication
- Admin dashboard
- Member dashboard
- Members management
- Subscriptions management
- Payments management
- Coaches management
- Attendance management
- AI reminder generator
