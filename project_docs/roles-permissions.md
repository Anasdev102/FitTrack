# Roles and Permissions

## Guest Permissions

Guest can:

- Access public website
- Register
- Login

Guest cannot:

- Access admin dashboard
- Access member dashboard

## Admin Permissions

Admin can:

- Access admin dashboard
- Manage members
- Manage subscriptions
- Manage payments
- Manage coaches
- Manage attendance
- Generate AI reminders
- View all data

Admin cannot:

- Access multi-tenant functionality because the project is not SaaS

## Member Permissions

Member can:

- Access member dashboard
- View own subscription
- View own payments
- View own attendance
- Update own profile

Member cannot:

- Access admin dashboard
- Manage members
- Manage subscriptions
- Manage payments
- Manage coaches
- Mark attendance
